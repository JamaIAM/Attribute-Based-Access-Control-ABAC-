<h1>Implementing ABAC Group Rules in Okta</h1>

Created group rules in Okta using user attributes to automatically assign users to the appropriate groups. By leveraging department-based attributes, access can be managed consistently and aligned with each user's role within the organization, performed on my computer.<br />

<h2>Environments and Technologies Used</h2>

- Okta

<h2>Operating Systems Used</h2>

- macOS Sequoia
